# AI × Design 2026

A conference presentation exploring lessons from the AI frontier — how design teams can navigate the rapid evolution of AI tools, processes, and culture.

## Talk structure

Five sections, three points each:

1. **Garbage in, garbage out** — organisational context, personal context, specificity & iteration
2. **Quality is a team sport** — slowing down to speed up, trust & safety, dogfooding
3. **Strategic alignment matters** — convergent thinking, knowing when not to use AI, hard problems unlock value
4. **Right tool for the job** — design process evolution, design systems as leverage, design commitments
5. **Get AI to do your dishes** — design systems & code, content design agents, the second brain

## Built with

- **React + TypeScript + Vite** — presentation app
- **Custom ASCII shader** — WebGL2 shader that renders images and patterns as animated ASCII art
- **CRT pixel shader** — green phosphor monochrome treatment for modal images and video
- **CSS CRT effects** — scanlines, vignette, and retro power-on/off animations

## Running locally

```bash
cd presentation/app
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
